SEND - IETF SEcure Neighbor Discovery (SEND) BoF/WG

This is a secondary home page for the IETF SEcure Neigborhood Discovery (SEND) Working Group. The purpose of this web page is to collect links to the relevant internet drafts, meeting minutes, etc. for easy access.

Co-chairs: James Kempf and Pekka Nikander

Mailing list

http://standards.ericsson.net/lists/ietf-send/threads.html

To subscribe to the mailing list, send a message to majordomo@standards.ericsson.se with a message containing

    subscribe ietf-send

in the body.
